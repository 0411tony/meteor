# meteor-platform

This package is added to every app by `meteor create`. It includes a set of common packages that most Meteor apps will find useful:

- `meteor`
- `webapp`
- `logging`
- `tracker`
- `session`
- `ddp`
- `mongo`
- `blaze`
- `spacebars`
- `templating`
- `check`
- `underscore`
- `jquery`
- `random`
- `ejson`

See the README of each individual package to learn more about it.
